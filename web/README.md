# FinYatra Web

Next.js application for [FinYatra](https://finyatra.com). See the [root README](../README.md) for full documentation, deployment, and contribution guidelines.

## Quick start

```bash
cp .env.example .env.local
npm install
npm run dev
```

Open http://localhost:3000

## Vercel

Set **Root Directory** to `web` and add env vars from `.env.example`.
